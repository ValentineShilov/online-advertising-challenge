Инструкция по воспроизведению решения:
1.  разместить train.csv, test.csv в папке со скриптами
2.  выполнить convert_data.py
3.  установить pypy
4.  выполнить pypy regr.py
5.  libffm:
    выполнить 
    cp train_p.csv libffm/tr.csv
    cp test_p.csv libffm/te.csv

    (или создать ссылки)
    cd libffm
    python run.py
6. выполнить run.pys
7. объеденить сабмиты pypy ensamble_submissions.py
8. загрузить final7.csv на kaggle
